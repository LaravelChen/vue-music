##技术栈
####1.采用Laravel5.3和Vue2.0完成的SPA应用，
####2.样式采用Muse_UI组件开发
####3.所有接口采用网易云音乐的Api.
####4.主要采用了Vue的全家桶(Vue+Vue-router+Vue-axios+Vuex+webpack+npm)
####5.本人采用的是Mac下面的valet的开发环境,比较方便.

##功能介绍
####1.支持搜索，单曲循环，列表循环，歌词滚动显示，音量调节，暂停，开始播放，下一首，上一首等.

##环境要求
>####Mac使用者最好搭建了homestead或者vlate环境
